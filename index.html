Etiquetas HTML Esenciales

<!DOCTYPE html>

La declaración <!DOCTYPE html> indica que se está utilizando HTML5

<html>

La etiqueta <html> es la raíz de un documento HTML y envuelve todo el contenido de la página (metadatos, hojas de estilo, scripts, elementos visibles).

Atributos globales:

Estos son algunos de los atributos que están disponibles para todos los elementos dentro de un documento HTML

  • lang -> Indica el idioma de la página.
  • class -> Especifica el nombre de clases de un elemento HTML (útil para identificación grupal).
  • id -> Especifica el nombre identificador de un elemento HTML (útil para identificación individual).
  • title -> Proporciona información extra al pasar el cursor sobre el elemento HTML.
  • hidden -> Oculta el elemento.
  • data-* -> Almacena información personalizada que un script pueda leer.

<head>

La etiqueta <head> contiene metadatos, enlaces a hojas de estilo, scripts y el título de la página.

<body>

El contenido visible de una página web se encuentra dentro de la etiqueta <body>

Atributos:

  • onload -> Le indica a la página qué script ejecutar al cargar.
  • onunload -> Le indica a la página qué script ejecutar al salir.

Encabezados

Los encabezados se utilizan para definir títulos y subtítulos en el contenido de una página. Van desde <h1> (más grande) hasta <h6> (más pequeño).

<h1>Encabezado H1</h1>

<h2>Encabezado H2</h2>

<h3>Encabezado H3</h3>

<h4>Encabezado H4</h4>

<h5>Encabezado H5</h5>
<h6>Encabezado H6</h6>

La lista ordenada <ol> </ol> se utiliza junto con su etiqueta hija <li> para agrupar y ordenar elementos.

Atributos

  • type (ol) -> Estilo de ordenamiento (decimal, alfabético, romano).
  • start (ol) -> Número de inicio de la lista.
  • reversed (ol) -> Orden inverso de numeración.
  • value (li) -> Especifica un valor para un elemento de la lista.

La lista desordenada <ul> </ul> se utiliza junto con su etiqueta hija <li> para agrupar elementos sin orden específico.

Párrafo <p>

Las etiquetas <p> </p> son utilizadas para agrupar líneas de texto que describen contenido de una página.

Imágen <img>

Las etiqueta <img> es utilizada para insertar una imágen en una página web.

Atributos:

  • src -> Especifica la ruta de la imágen.
  • alt -> Texto alternativo que describe la imágen.
  • width -> Ancho de la imágen en píxeles o porcentaje.
  • height -> Alto de la imágen en píxeles o porcentaje.
  • loading -> Modo de carga de la imágen (lazy o eager).

Enlace <a>

Las etiquetas <a> </a> son utilizadas para definir un enlace

Atributos:

  • href -> Destino del enlace.
  • target -> Indica dónde se abre el enlace (_blank, _parent, _self, _top).
